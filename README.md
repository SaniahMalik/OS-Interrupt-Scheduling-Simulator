# OS Interrupt Scheduling Simulator 🧠💻

A modern, dark-themed GUI dashboard built using Python and PyQt5 to visualize core Operating System concepts like Interrupt Handling, Preemption, and CPU Scheduling.

## ⚡ Features Built
- **Real-Time Preemption Engine:** Automatically halts low-priority tasks when an interrupt storm occurs.
- **Context Switching Visualizer:** Animates CPU state transitions (Idle, Running, Switching).
- **Dynamic Gantt Chart:** Plots a live time-distribution grid for process tracking.

## 🛠️ Tech Stack
- Python 3.x
- PyQt5 (GUI Framework)
